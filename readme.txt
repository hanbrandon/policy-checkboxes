Adds checkboxes for privacy policy and terms and conditions to the WooCommerce registration form and allows users to set the pages in the WordPress dashboard.

== Description ==

This plugin adds checkboxes for the privacy policy and terms and conditions to the WooCommerce registration form, requiring users to agree to these policies before creating an account. The privacy policy and terms and conditions pages can be set in the WordPress dashboard.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-policy-checkboxes` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the General Settings page in the WordPress dashboard and set the pages to use as the privacy policy and terms and conditions.

== Frequently Asked Questions ==

= How do I set the privacy policy and terms and conditions pages? =

To set the privacy policy and terms and conditions pages, navigate to the General Settings page in the WordPress dashboard and select the pages from the dropdown menus provided.

= Can I customize the checkboxes? =

Yes, you can customize the checkboxes by modifying the `woocommerce_policy_checkboxes()` function in the plugin code.
